# azure-cli-powershell

Dockerfile for azure-cli with PowerShell on Ubuntu Linux (Xenial - 16.04)

## Usage

```sh
docker run -it matsuu/azure-cli-python sh
```

## Build

```sh
docker build -t matsuu/azure-cli-python .
```

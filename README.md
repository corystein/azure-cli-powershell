# azure-cli-powershell

Dockerfile for azure-cli with PowerShell on Ubuntu Linux (Xenial - 16.04)

## Usage

```sh
docker run -it corystein/azure-cli-powershell sh
```

## Build

```sh
docker build -t corystein/azure-cli-powershell .
```
